# 01 - 项目需求文档（Requirements）

**项目名称**：Koala - 你的下一代 AI 学习助手  
**版本**：v0.4（Vertex AI Vector Search + Firebase Anonymous Auth + 一次性生成关卡）  
**编写日期**：2026年3月  
**作者**：Johnson  
**目标**：为 Gemini Live Agent Challenge 提交一个沉浸式、多模态、实时交互的 AI 学习代理，强调深度自适应反馈 + 自定义知识库 + 闯关游戏化体验。

## 1. 项目愿景与核心价值

Koala 不是传统课程平台，而是一个**温柔的 AI 学习伙伴**，像一只抱着树枝的考拉，静静陪你慢慢成长。

核心价值：
- **深度自适应**：根据用户真实反馈实时调整课程大纲和难度
- **自定义知识库**：用户上传资料 → 构建专属 RAG 知识库，答案必须严格引用来源
- **沉浸式闯关**：类似 Duolingo + Brilliant 的多模态关卡（文本/语音/图像/SVG动画/题型混合）
- **实时语音互动**：全程监听用户自言自语（需授权），即时给出反馈
- **完全免费**：暂无付费墙，专注学习体验

## 2. 目标用户

- **主要用户**：对商业哲学、认知科学、个人成长、AI 时代生存感兴趣的学习者
- **用户画像**：
  - 年龄：18–40 岁
  - 背景：零基础到有一定基础但想系统化
  - 痛点：传统课程太水/太难/不贴合兴趣、缺乏深度反馈、容易半途而废
  - 期望：像朋友聊天一样学、内容来自可信来源、学习过程有趣不焦虑

## 3. 用户故事（User Stories）- MVP 核心

### 3.1 课程创建与知识源
- As a 用户，我希望选择一个学习主题（或自定义输入），以便开始个性化课程。
- As a 用户，我希望上传 PDF/网页链接作为知识来源，以便系统构建专属知识库。
- As a 用户，如果不上传知识源，系统自动联网搜索权威内容，并严格引用来源。

### 3.2 学习需求了解
- As a 用户，我希望系统先询问我的学习基础、目的、期望难度、期望节奏，以便生成合适的大纲。

### 3.3 课程大纲生成
- As a 用户，我希望系统生成细粒度课程大纲（章节 + 关卡），以便我能预览整体路径。
- As a 用户，我希望每完成一个关卡后，系统根据我的反馈动态调整后续关卡大纲。

### 3.4 闯关式学习体验
- As a 用户，我希望每个关卡有 20+ 个 step，融合多种模态呈现：
  - 文本
  - 图像/SVG 动画辅助理解（Gemini 直接生成）
  - 交互题型：选择题、填空题、判断题、问答题
- As a 用户，我希望关卡内容**一次性生成全部 20+ step**，前端分页加载，无需等待。
- As a 用户，我希望关卡末尾系统询问我的反馈（文本 + 实时语音自言自语），以便调整后续内容。

### 3.5 实时语音反馈
- As a 用户，我希望全程开启麦克风（需明确授权），随时自言自语，系统立即监听并给出反馈。
- As a 用户，如果拒绝录音，系统降级为纯文本反馈模式。

### 3.6 学习记录与游戏化
- As a 用户，我希望系统记录我的学习进度（课程、关卡完成情况、作答记录、反馈）。
- As a 用户，我希望有 XP 积分系统：
  - 答对 1 题 = **10 XP**（固定值）
  - 每个关卡 20+ step，满分 200+ XP/关
  - 连续答对 3 题 = 额外 10 XP 奖励
- As a 用户，我希望能随时删除课程，清空所有相关记录。

## 4. 非功能需求

- **性能**：关卡加载 < 5 秒，语音反馈延迟 < 2 秒
- **网络**：本地开发也必须联网，使用 Firestore 存储
- **隐私**：用户授权录音；数据关联设备 ID；支持删除所有记录
- **认证**：**游客模式**，使用 Firebase Anonymous Auth，数据关联匿名用户 ID
- **游客认证**：Firebase Anonymous Auth → 生成匿名用户 ID → 关联所有用户数据
- **安全性**：知识库内容严格引用来源，避免幻觉
- **兼容性**：浏览器支持（Chrome/Edge 优先）；移动端响应式
- **部署**：必须运行在 Google Cloud（Cloud Run + Vertex AI + Firestore）

## 5. MVP 范围（比赛提交版本）

必须实现：
- 上传知识源 → RAG 构建知识库 → 严格引用
- 询问需求 → 生成大纲 → 多模态关卡（至少文本 + 图像 + 题型 + 语音）
- 实时语音自言自语反馈（Gemini Live）
- XP + 音效（固定：答对 1 题 = 10 XP）+ 删除课程

可后续迭代：
- 更复杂的游戏化（等级、徽章）
- 导出学习总结
- 多人协作学习

## 6. 成功指标（比赛视角）

- Demo 视频能展示：上传 PDF → 知识库构建 → 实时语音反馈 → 大纲动态调整
- 评委能感受到“哇塞”：从静态聊天到沉浸式、实时、个性化学习体验
- 项目运行稳定、无明显 bug，部署在 Google Cloud
